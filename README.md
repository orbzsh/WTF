###列表
	无序列表，前面加上如下:
	-
	+
	*

	有序列表,前面加上如下:
	1.
	2.
	3.


###链接
```
[](),方括号里是显示的文本,圆括号里是链接的地址
![](),前面的！号表示图片链接
![](http://ww4.sinaimg.cn/bmiddle/aa397b7fjw1dzplsgpdw5j.jpg)
```
