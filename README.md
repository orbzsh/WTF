------------------------------------
###列表
	无序列表，前面加上如下:
	-
	+
	*

	有序列表,前面加上如下:
	1.
	2.
	3.
1. 有序列表1
2. 有序列表2

*	无序列表1
-	无序列表2
+	无序列表3

###引用
```
使用>后面跟上引用语句，可以换行等，注意>号后面要有一个空格
```
> 引用：人生不止眼前的苟且，还要有诗和远方

>> 你走，我不送你。
>> 你来，无论多大风多大雨，我要去接你。

###粗体斜体
```
用两个*包含一段文本就是粗体,用一个*包含文本就是斜体
```
